ATM Console Application

This project is a Java-based ATM simulation that allows users to check balance, deposit money, and withdraw cash through a menu-driven console interface. It includes PIN verification with limited attempts, making it a simple and effective project for learning core Java and OOP basics.

Features

PIN authentication (3 attempts)
Check account balance
Deposit money
Withdraw money with validation
Menu-driven console interface

Technologies Used

Java
OOP Concepts
Scanner for user input
